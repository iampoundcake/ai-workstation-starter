honest disclosure

It's a starter, not a product. Here's what that buys you, and what it doesn't.

// what you get
• A vault structure that survives whatever you throw at it
• Clickable dashboard with 4 themes (workshop · daylight · terminal · studio)
• Four working skills out of the box
• An interview that builds skills tailored to your real work
// what you don't get
• Support (it's a personal repo)
• Updates (probably)
• Anyone else's opinions about how you should work
• A SaaS dashboard or hosted version
verdict · honest06 / 08
install · two commands

want to try it?

Two commands, then a guided interview. Five minutes from clone to first skill.

# 1. clone the starter
$ git clone https://github.com/iampoundcake/ai-workstation-starter ~/ai-workstation

# 2. run setup (asks for vault path + theme)
$ cd ~/ai-workstation && ./setup.sh

# 3. inside Claude Code, start the interview
$ /aiworkstation init
